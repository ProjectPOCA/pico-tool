# pico tool — Privacy Statement

pico tool runs entirely on your computer.

- **No accounts, no analytics, no telemetry.** The app does not collect, transmit, or store any
  personal information.
- **Saved configurations** (My Pico) are stored locally in your user data folder and never leave
  your machine.
- **Network access** is used for exactly one thing: checking GitHub Releases for a newer version
  of the app. No other connections are made.
- **USB access** is used only to communicate with the Raspberry Pi Pico you choose to flash.
